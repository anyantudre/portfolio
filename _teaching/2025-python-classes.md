Delivered weekly Python programming classes to **60+ students** through a community initiative from March to July 2025.

## Course Content

- Python fundamentals: variables, data types, control flow
- Data structures: lists, dictionaries, sets, tuples
- Functions and modules
- Object-oriented programming
- File handling and exceptions
- Introduction to NumPy and Pandas

## Teaching Approach

- Hands-on coding exercises each week
- Real-world projects to apply concepts
- Interactive Q&A sessions
- Code review and feedback
